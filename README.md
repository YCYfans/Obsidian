# 韦石胜的个人 Obsidian 知识库

这是一个用于个人知识管理与思考的 Obsidian 数字花园。本仓库记录了我在学习、工作和生活中的各种笔记、想法和项目。

## 仓库结构

```
.
├── 📁 00-Inbox              # 临时收集区
├── 📁 01-DailyNotes         # 每日笔记
├── 📁 02-Areas              # 领域知识（长期关注的主题）
├── 📁 03-Projects           # 项目相关笔记
├── 📁 04-Resources          # 资源收集
├── 📁 05-Archive            # 归档文件
└── 📁 Templates             # 笔记模板
```

## 主要功能

### 1. 知识管理
- **领域笔记**：按主题分类的专业知识积累
- **项目笔记**：项目相关的规划、进度和总结
- **文献笔记**：阅读书籍、论文的摘要和思考

### 2. 日常工作流
- **每日笔记**：记录当天的想法、任务和事件
- **周回顾**：每周总结和计划
- **月度复盘**：每月反思和调整

### 3. 思考工具
- **概念图**：使用 Canvas 进行可视化思考
- **闪念笔记**：快速记录瞬间想法
- **永久笔记**：经过整理和连接的核心知识

## 使用工具

- **Obsidian**：主要笔记工具
- **Git**：版本控制和同步
- **插件生态**：根据需求扩展功能

## 核心原则

1. **原子化**：每个笔记聚焦一个核心概念
2. **连接性**：通过双向链接建立知识网络
3. **渐进式**：从临时笔记到永久笔记的渐进整理
4. **实用性**：知识服务于实际工作和思考
