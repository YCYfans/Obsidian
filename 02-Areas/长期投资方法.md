## 1. 长期投资的核心理念

### 1.1 什么是长期投资？

长期投资不是"买了不动"，而是**买对了、理解透了、拿得住**。时间框架一般在 3~10 年以上。

核心本质：**用时间换取企业价值的增长，让复利帮你做事。**

### 1.2 复利的魔力

爱因斯坦说："复利是世界第八大奇迹，懂的人赚它，不懂的人付它。"

| 年化收益率 | 10年 | 20年 | 30年 |
|-----------|------|------|------|
| 5%        | 1.63x | 2.65x | 4.32x |
| 10%       | 2.59x | 6.73x | 17.4x |
| 15%       | 4.05x | 16.4x | 66.2x |
| 20%       | 6.19x | 38.3x | 237x |

**关键点：** 年化多5%，30年后差距是5倍以上。所以选择好标的、减少摩擦成本（税、手续费）非常重要。

### 1.3 时间是优质资产的朋友

- 烂公司：持有越久越亏，价值持续流失
- 平庸公司：持有很久回到原点
- 优质公司：时间让价值不断累积，熊市只是短暂压低价格

> 💡 核心判断：这家公司10年后还会在吗？会比现在更大吗？

### 1.4 价值投资 vs 投机

| 维度 | 价值投资 | 投机 |
|------|---------|------|
| 思维方式 | 买的是公司的一部分 | 买的是一张价格波动的票 |
| 持有时间 | 年为单位 | 天/周 |
| 分析对象 | 企业基本面 | 价格图形、资金流 |
| 胜率来源 | 企业价值增长 | 博弈对手犯错 |
| 情绪管理 | 需要极大耐心 | 需要极快反应 |

### 1.5 三个核心支柱

```
长期投资 = 选对公司 × 合理价格买入 × 耐心持有
```

缺少任何一条都会失败：
- 好公司 + 价格太贵 → 长期回报差（如2021年高点买白酒）
- 便宜价格 + 烂公司 → 价值陷阱（如某些ST股）
- 好公司 + 好价格 + 频繁交易 → 耗费手续费、踩错时点

## 2. 巴菲特/芒格的投资哲学

### 2.1 核心原则总结

| 原则 | 巴菲特原话/行为 | 实际含义 |
|------|--------------|---------|
| 安全边际 | "第一条：不要亏钱。第二条：不要忘记第一条" | 买入价格要低于内在价值，留出缓冲 |
| 能力圈 | "我不做科技股，因为我看不懂" | 只在自己理解的行业投资 |
| 护城河 | "我寻找周围有护城河的城堡" | 企业要有持久的竞争优势 |
| 集中持仓 | "分散是无知的保护，聪明人不需要它" | 真正看懂的机会，重仓买入 |
| 长期持有 | "最喜欢的持有期是永远" | 找到好公司，不要因为短期波动卖出 |
| 等待机会 | "市场先生是你的仆人，而非你的向导" | 等好价格，不受市场情绪影响 |

### 2.2 市场先生理论

格雷厄姆的比喻：市场先生每天来找你出价买你的股票，有时候他情绪高涨报高价，有时候他情绪低落报低价。

**你的选择：**
- 市场先生报价低 → 买入（用他的恐慌赚钱）
- 市场先生报价高 → 卖出或观望（用他的贪婪赚钱）
- 大多数时候 → 忽视他（你不需要每天对他的报价做出反应）

### 2.3 芒格的多元思维模型

芒格强调要建立"思维格栅"，用不同学科的工具分析投资：

- **心理学**：避免认知偏差（确认偏误、从众心理）
- **统计学**：理解概率和基础概率（不要被小样本欺骗）
- **物理学**：临界点效应（企业规模到某点后加速增长）
- **生物学**：生态位和竞争（行业生态如何影响企业）
- **经济学**：激励机制（管理层利益是否与股东一致）

### 2.4 巴菲特选股的实际流程（简化版）

```
1. 这个行业我能看懂吗？  → 否：跳过
2. 这家公司有护城河吗？  → 否：跳过
3. 管理层诚实且有能力吗？→ 否：跳过
4. 当前价格有安全边际吗？→ 否：等待
5. 都满足 → 重仓买入，长期持有
```

### 2.5 几个被忽视的巴菲特观点

- **不要过度分散**：持有20只股票往往比持有5只表现差，因为你对5只的研究更深
- **现金也是仓位**：高估时持有现金等待机会，不是胆小，是智慧
- **避免短期预测**："我从来不预测市场短期走势，那是浪费时间"
- **买股票就是买公司**：如果你不愿意持有这家公司10年，就不要买10分钟

## 3. 指数基金长期投资策略

### 3.1 为什么指数基金适合大多数人？

**残酷的现实：**
- 90% 的主动基金经理长期无法跑赢指数
- A股散户平均年亏损约 15%-20%（交易摩擦 + 情绪决策）
- 指数基金持有成本低（年费 0.1%-0.5% vs 主动基金 1.5%+）

**结论：** 普通人指数基金定投，长期大概率跑赢 80% 的专业投资者。

### 3.2 A股主要指数对比

| 指数 | 代码 | 成分股数量 | 特点 | 适合谁 |
|------|------|-----------|------|--------|
| 沪深300 | 000300 | 300只 | A股大盘蓝筹，代表性强 | 入门首选 |
| 中证500 | 000905 | 500只 | 中盘股，弹性更大 | 进阶 |
| 中证1000 | 000852 | 1000只 | 小盘股，波动大 | 高风险偏好 |
| 创业板指 | 399006 | 100只 | 成长型科技为主 | 看好科技长线 |
| 恒生指数 | HSI | ~80只 | 港股蓝筹 | 港股配置 |
| 纳斯达克100 | NDX | 100只 | 美国科技巨头 | 美股配置 |

### 3.3 普通定投策略（傻瓜版）

**操作：**
- 每月固定日期（如发工资后第2天）
- 买入沪深300/中证500 ETF
- 不管涨跌，机械执行
- 至少坚持5年

**为什么有效：**
- 下跌时买得多（降低成本）
- 上涨时买得少（避免追高）
- 时间足够长，自然覆盖多个市场周期

### 3.4 估值定投策略（进阶版）

根据市场估值调整定投金额，便宜时多买，贵时少买。

**PE（市盈率）参考标准（沪深300）：**

| PE区间 | 市场状态 | 操作建议 | 月投金额倍数 |
|--------|---------|---------|------------|
| < 10   | 极度低估 | 大幅加仓 | 3x |
| 10-13  | 低估     | 加仓     | 2x |
| 13-16  | 合理     | 正常定投 | 1x |
| 16-20  | 略高     | 减少定投 | 0.5x |
| > 20   | 高估     | 暂停/减仓| 0x |

> 🔗 查估值：**中证指数官网**、**集思录**、**且慢** 都有实时PE/PB数据

### 3.5 再平衡策略

**目标配置示例（中等风险）：**
```
权益类（股票/基金）：60%
固定收益（债券/货基）：30%
另类（黄金/REITs）：10%
```

**再平衡时机：**
- **时间触发**：每半年或每年强制再平衡一次
- **偏差触发**：某类资产偏离目标超过 ±10%，就调整回来
- **实际操作**：不要卖出上涨的，而是用新资金补充落后的部分

**例子：**
> 原本60%股票，牛市后变成75%，此时减少股票买入，多买债券/黄金，把比例调回60%

## 4. 如何选股

### 4.1 护城河理论

护城河 = 竞争优势的可持续性。分5种：

| 护城河类型 | 说明 | A股典型例子 |
|-----------|------|-----------|
| **成本优势** | 同质产品成本远低于竞争者 | 海螺水泥、万华化学 |
| **网络效应** | 用户越多产品越有价值 | 微信（腾讯）、支付宝 |
| **无形资产** | 品牌、专利、特许经营权 | 茅台（品牌）、片仔癀（药方） |
| **转换成本** | 用户切换成本极高 | 用友网络（ERP系统）、恒瑞医药 |
| **高效规模** | 市场容量只够一家盈利 | 部分区域性公用事业 |

**判断护城河强度的问题：**
1. 如果一个资本雄厚的竞争者进入，你的公司还能活得好吗？
2. 公司的定价权在哪里？能否提价而不失去客户？
3. 过去10年ROE（股本回报率）是否持续 >15%？

### 4.2 核心财务指标

**必看指标（从简到深）：**

| 指标 | 参考值 | 说明 |
|------|--------|------|
| ROE（净资产回报率）| >15%，持续10年 | 衡量公司赚钱效率，越高越好 |
| 毛利率 | >40% 为优 | 产品竞争力的直接体现 |
| 净利润增速 | >15% | 成长性 |
| 资产负债率 | <60%（非金融类）| 财务健康度 |
| 自由现金流 | 与净利润接近 | 验证利润是否真实 |
| PE（市盈率）| 行业比较 | 衡量贵不贵，需结合行业 |
| PB（市净率）| <3 为基础参考 | 适合重资产公司 |

**红旗警示（看到这些要小心）：**
- 应收账款增速 >> 营收增速（可能是虚假收入）
- 净利润高但现金流为负（利润不真实）
- 频繁融资/增发（管理层在"圈钱"）
- 管理层持续大规模减持
- 关联交易复杂、审计意见保留

### 4.3 行业分析框架

选行业比选个股更重要，因为70%的个股涨跌跟随行业。

**优质行业特征：**
- 需求稳定（食品饮料、医疗）或长期增长（新能源、科技）
- 行业集中度提升（强者恒强）
- 政策支持或至少不打压
- 进入壁垒高（专利、资质、规模）

**A股行业长期回报（粗略参考）：**
```
★★★★★ 白酒/消费品（茅台效应，但估值贵）
★★★★  医药/医疗器械（老龄化驱动）
★★★★  新能源（政策+全球趋势）
★★★   银行/保险（低估但增速慢）
★★    地产（政策风险大，慎重）
★     钢铁/煤炭（周期性，需要择时）
```

## 5. 如何分析一家公司

### 5.1 商业模式分析

**核心问题：这家公司靠什么赚钱？**

分析维度：
1. **客户是谁？** 个人消费者 vs 企业 vs 政府（稳定性不同）
2. **卖的是什么？** 一次性产品 vs 持续订阅/复购
3. **利润从哪来？** 规模效应 vs 定价权 vs 成本控制
4. **增长从哪来？** 新客户 vs 老客户复购 vs 提价

**商业模式优劣对比：**

| 类型 | 特点 | 例子 |
|------|------|------|
| 消耗品/复购模式 | 客户反复购买，现金流稳定 | 茅台、海天味业 |
| 订阅/会员模式 | 现金流可预测，客户黏性高 | SaaS软件公司 |
| 平台模式 | 网络效应，赢者通吃 | 腾讯、阿里 |
| 重资产模式 | 资本开支大，回报慢 | 钢铁、地产 |
| 项目制模式 | 收入不连续，管理难 | 建筑工程类 |

### 5.2 竞争优势检验

**"护城河测试"3个问题：**

1. **涨价测试**：公司能涨价 10% 而不损失客户吗？（茅台能，超市不能）
2. **跟随测试**：竞争对手能在3年内复制它的核心优势吗？
3. **衰退测试**：如果整体经济下滑，这家公司还能维持盈利吗？

### 5.3 管理层评估

好的管理层是看不见的护城河：

**看什么：**
- **诚信**：历年年报承诺是否兑现？是否有操控业绩嫌疑？
- **能力**：ROE 趋势、战略执行力、危机应对
- **利益一致**：管理层持股比例是否高？薪资与业绩挂钩吗？
- **分配决策**：分红 vs 回购 vs 盲目扩张？

**A股额外关注：**
- 大股东占款（用公司钱给大股东"借钱"）
- 频繁并购（很多是利益输送）
- 股权质押比例（质押超过 50% 要警惕爆仓风险）

## 6. 长期持有的心态建设

### 6.1 市场波动是常态，不是异常

**A股历史数据告诉你：**
- 从1990年到2025年，上证综指上涨了约 50 倍
- 期间经历了 **12次** 超过 30% 的大跌
- 每次大跌时"这次不一样"的声音都很响
- 但每次都涨回来了（前提是买的是好公司或指数）

| 时间 | 大跌幅度 | 恢复时间 |
|------|---------|---------|
| 2001-2005 | -55% | 约2年 |
| 2007-2008 | -70% | 约3年 |
| 2015-2016 | -45% | 约2年 |
| 2021-2024 | -40%+ | 进行中 |

**结论：** 熊市不是终点，是优质股票打折促销的时候。

### 6.2 对抗情绪的实用方法

**1. 写投资日记**
记录每次买入的理由，熊市时重读，防止因恐慌卖出好资产。

**2. 不要每天看账户**
设定规则：每周最多看一次账户，季报/年报出来时才做决策。

**3. 计算公司价值，而非股价**
股价下跌 30% 时问自己：**公司的实际业务变差了吗？** 如果没有，价格下跌是机会。

**4. 构建"熊市自救清单"**
预先准备好熊市时要读的材料：
- 巴菲特历年股东信
- 你最初买入这家公司的分析笔记
- 复利表格（想想10年后是什么样）

**5. 永远保留 20%-30% 的现金**
不满仓，熊市才有子弹。有子弹就不怕下跌，甚至期待下跌。

### 6.3 几个反常识但重要的认知

- **下跌 50% 需要上涨 100% 才能回本** → 控制亏损是第一优先级
- **频繁交易必输** → 每次买卖你都要付出手续费和税，市场长期是零和游戏，成本越低胜率越高
- **看不懂不买** → 看不懂的机会不是机会，是陷阱
- **好公司也有买贵的时候** → 2021年茅台300倍PE，之后跌了60%；好公司≠可以任何价格买

## 7. 仓位管理与资产再平衡

### 7.1 仓位管理基本原则

**核心原则：不要让任何单一押注毁掉你的组合**

```
单只股票仓位 ≤ 总资产的 20%（集中型投资者）
单只股票仓位 ≤ 总资产的 10%（稳健型投资者）
总股票仓位  ≤ 个人可承受的最大亏损 / 最大可能亏损幅度
```

**例子：**
> 总资产 100 万，心理承受上限是亏损 30 万（30%），
> 假设股市最大可能跌 50%，
> 那股票仓位上限 = 30% / 50% = 60 万（60%）

### 7.2 分阶段建仓（不要一把梭）

| 建仓阶段 | 触发条件 | 投入资金 |
|---------|---------|---------|
| 第一仓  | 看好这家公司，价格合理 | 计划资金的 1/3 |
| 第二仓  | 股价下跌 15%-20% | 计划资金的 1/3 |
| 第三仓  | 股价再跌 20% 或等到季报确认基本面没变 | 计划资金的 1/3 |

好处：避免踩在阶段性高点，同时防止好公司一直涨而错过机会。

### 7.3 资产配置参考模型

**根据年龄和风险承受能力调整：**

| 年龄段 | 权益类 | 固收类 | 现金/其他 |
|-------|-------|-------|---------|
| 20-30岁 | 70-80% | 15-20% | 5-10% |
| 30-40岁 | 60-70% | 20-30% | 10% |
| 40-50岁 | 50-60% | 30-40% | 10-20% |
| 50岁以上 | 30-50% | 40-50% | 10-20% |

> ⚠️ 以上只是参考，个人的稳定收入、负债情况、风险承受能力才是真正决定因素

### 7.4 什么时候减仓/卖出？

**应该卖出的情况：**
1. 买入逻辑被打破（不是价格下跌，而是基本面变了）
2. 有更好的机会，置换更高性价比标的
3. 估值严重高估，已远超合理区间（如PE > 历史95%分位）
4. 仓位比例严重失衡，需要再平衡

**不应该卖出的情况：**
- 价格下跌了但基本面没变
- 别人在卖，感觉慌了
- 短期业绩不好（季度波动很正常）
- 宏观经济暂时不好

## 8. A股长期投资的特殊考量

### 8.1 A股市场特点

| 特点 | 说明 | 应对策略 |
|------|------|---------|
| 散户主导 | 散户占成交量约 70%，情绪化波动大 | 反向操作，贪别人恐惧时，恐别人贪婪时 |
| 政策市 | 政策对市场影响远超基本面 | 关注政策方向，不硬扛政策 |
| 退市机制不完善 | 烂公司不容易退市，长期占用资源 | 不碰ST，不碰负面缠身的公司 |
| 内幕交易较多 | 信息不对称严重 | 只买流动性好的大市值公司，避开小盘股信息陷阱 |
| 牛短熊长 | A股平均牛市持续约1.5年，熊市约3年 | 做好长期等待的心理准备 |

### 8.2 政策风险管理

**A股的政策逻辑：**
- 监管方向 = 国家战略方向（新能源、半导体、国产替代）
- 被打压的行业短期内很难翻身（如2021年的教育、互联网）
- 政策利好的行业往往提前在股价中反映

**应对原则：**
1. **顺势而为**：国家重点支持的行业，长期有结构性机会
2. **不硬扛监管**：被政策打压的行业（无论多便宜）先观望
3. **分散行业**：不要把所有资金押在单一政策敏感行业

**当前（2025年）政策方向参考：**
- ✅ 新能源/储能/电网
- ✅ 半导体/国产替代
- ✅ 医疗/创新药（老龄化）
- ✅ 高端制造/出口
- ⚠️ 互联网平台（监管松动但不确定）
- ❌ 学科教育（政策明确限制）
- ❌ 地产（政策救市但长期逻辑弱）

### 8.3 A股特有机会

- **北向资金**：外资是A股最理性的增量资金，大规模流入时往往是布局机会
- **国企改革**：部分央企/国企被低估，改革后ROE提升空间大
- **港股通**：优质港股估值比A股低 30%-50%，可以通过港股通配置同类资产
- **季节性规律**：A股有一定季节性规律（春季躁动、年底效应），可作为择时辅助参考

### 8.4 税务和成本

| 成本项 | A股 | 港股 | 美股 |
|-------|-----|------|------|
| 印花税 | 卖出 0.1% | 卖出 0.1% | 无 |
| 佣金 | 约 0.02-0.03% | 约 0.03-0.1% | 约 $1/笔 |
| 红利税 | 持有<1年 20%，>1年免税 | 20% | 20% |

**建议：** 长期持有股息股，务必持有满一年享受免税，否则分红收益会大幅缩水。

## 9. 实际操作案例

> ⚠️ 以下为假设场景，仅用于说明方法，非投资建议。

### 案例一：指数基金定投（稳健型）

**背景：** 小李，30岁，月薪 2 万，每月可投资 3000 元，目标是 20 年后退休。

**策略：**
- 每月 1 号，买入沪深300 ETF（代码：510300）
- 使用估值定投：查询沪深300 PE，低于 13 倍时加倍买入
- 每年 12 月做一次再平衡，维持 60% 股票/30% 债券/10% 黄金

**20年模拟结果（假设年化8%）：**
```
月投 3000 元 × 240 个月 = 累计投入 72 万
年化 8% 复利 → 最终约 177 万
实际收益约 105 万（1.46 倍投入）
```

---

### 案例二：个股长期持有（白酒行业）

**背景：** 老王，2018 年研究贵州茅台，决定长期持有。

**分析过程：**

| 维度 | 评估 |
|------|------|
| 护城河 | ★★★★★ 品牌+文化+产能稀缺，无可替代 |
| ROE | 持续 30%+ 超过10年 |
| 毛利率 | 约 92%，极高 |
| 管理层 | 国企背景，稳定，分红慷慨 |
| 风险 | 政策限制三公消费、食品安全 |

**2018 年操作：**
- 茅台当时约 560 元/股，PE 约 25 倍（历史中位）
- 分三批买入：560 元、520 元、490 元（等跌）
- 总成本约 523 元

**2021 年高点：** 茅台达到 2600 元，涨幅近 5 倍，PE 高达 60 倍以上

**决策点：** 估值过高，超出历史95%分位，减仓 50%，落袋为安

**教训：**
- 好公司也会被炒到疯狂高估
- 高点没有全卖，是因为看不准顶，减仓而非清仓是更务实的做法
- 剩余 50% 继续持有，等待合理估值再加回

---

### 案例三：熊市加仓（心理测试）

**背景：** 小张，2021 年底买入某医药 ETF，之后经历医药行业持续下跌 40%。

**考验：**
- 2022 年，医药 ETF 从 2 元跌到 1.2 元（-40%）
- 市场充斥"医药已死"声音
- 小张账面浮亏 10 万

**正确应对：**
1. 重新阅读买入逻辑：老龄化趋势没有改变，行业只是估值修复
2. 核查 ETF 成分股：龙头公司基本面没有变差
3. 分析下跌原因：集采政策 + 美股科技股下跌引发调整
4. 决策：用剩余现金在 1.2 元区间再次买入（摊低成本至 1.5 元）
5. 持续定投，不看账户

**结果（2024年）：** 医药 ETF 反弹至 1.8 元，总持仓接近回本，并继续持有

**核心经验：**
> 下跌时能买入，才是真正的价值投资。下跌时恐慌卖出，之前做的所有研究都白费了。

## 10. 常见误区和陷阱

### 10.1 思维误区

| 误区 | 危害 | 纠正方法 |
|------|------|---------|
| **追涨杀跌** | 永远买贵卖便宜，必亏 | 建立买入纪律，跌了才是机会 |
| **频繁换股** | 手续费侵蚀+错过复利 | 换股前问：新的比旧的好多少？ |
| **把亏损股留着，卖掉盈利股** | 养着"烂孩子"，送走"好孩子" | 决策基于未来价值，不基于成本价 |
| **相信"内部消息"** | 消息散播时你已是接盘侠 | 只信公开披露的财务数据 |
| **过度自信** | 认为自己比市场更聪明 | 定期复盘，记录自己的错误判断 |
| **锚定效应** | "我在30元买的，跌到15元还贵" | 忘掉成本价，只看当前价值 |

### 10.2 操作陷阱

**陷阱一：价值陷阱**
> 看起来很便宜（PE很低），但公司持续下滑，永远涨不回去。
> **典型**：很多传统行业的低PE股，如纸媒、传统零售
> **识别**：ROE 持续下滑，行业天花板已到，管理层无创新

**陷阱二：高股息陷阱**
> 高分红 → 股价下跌 → 实际收益为零甚至亏损
> **典型**：部分煤炭、地产股，股息率看似很高，但股价腰斩
> **识别**：分红是否来自自由现金流？还是借债分红？

**陷阱三：行业景气陷阱**
> 行业景气高峰时买入，景气一旦下行，估值和盈利双杀
> **典型**：2021年新能源高点、2007年钢铁高点
> **识别**：当所有人都在讨论这个行业，大概率已经过热

**陷阱四：故事陷阱**
> 被公司的"转型故事"吸引，实际上是用PPT掩盖糟糕的基本面
> **典型**：很多A股"主题投机股"
> **识别**：有故事但没有利润增长，看故事不如看财务

**陷阱五：幸存者偏差**
> 只看到成功的长期投资案例（茅台），忘了同期还有无数跌90%的股票
> **纠正**：不仅要研究成功案例，更要研究失败案例的共同特征

### 10.3 心理陷阱

- **FOMO（错过恐惧）**：别人都在赚钱我也要赶紧买 → 往往追高
- **确认偏误**：只看支持自己判断的信息，忽视反面证据
- **近因偏差**：最近涨了就觉得会继续涨，最近跌了就觉得会继续跌
- **赌徒谬误**：跌了很多"应该要涨了" → 市场没有欠你的

## 11. 长期投资者的自我修养清单

> 把这个清单打印出来，每半年对照一次。

### 📋 买入前检查

- [ ] 我能用一句话解释这家公司/基金为什么值得买吗？
- [ ] 我理解它的商业模式和主要风险吗？
- [ ] 当前估值是否在合理区间（不是买便宜，是买合理）？
- [ ] 我是否留有足够的安全边际（通常 20-30%）？
- [ ] 这笔钱是否是我 3 年内不需要用到的？
- [ ] 如果明天跌 30%，我能淡定继续持有甚至加仓吗？
- [ ] 我是否因为别人推荐/市场热门而买，还是因为自己研究？
- [ ] 单只标的是否控制在总仓位的 10-20% 以内？

### 📋 持有期间检查（每季度）

- [ ] 这家公司的买入逻辑有没有被破坏？
- [ ] 最新财报是否符合预期？有没有负面变化？
- [ ] 行业整体趋势是否发生了重大变化？
- [ ] 管理层有没有异常动作（大规模减持、违规）？
- [ ] 我的资产配置比例是否需要再平衡？
- [ ] 我有没有因为市场短期波动做出非理性操作？

### 📋 卖出前检查

- [ ] 卖出原因是基本面变化，还是价格下跌带来的恐惧？
- [ ] 如果卖出，这笔钱要去哪里？是否有更好的机会？
- [ ] 卖出后是否会有后悔的可能（涨了怎么办/跌了怎么办）？
- [ ] 当前估值是否已经严重高估（PE > 历史90%分位）？
- [ ] 我的决定是否经过至少24小时的冷静期？

### 📋 年度复盘检查

- [ ] 今年的投资决策中，哪些是对的？原因是什么？
- [ ] 哪些决策是错的？我是否从中学到了教训？
- [ ] 我是否遵守了自己的投资纪律，没有被情绪左右？
- [ ] 我的整体资产配置是否还符合我的年龄/风险承受能力？
- [ ] 我的投资知识是否比一年前提高了？

### 🧠 投资者心态准则

1. **股价下跌是礼物，不是灾难**（前提是买了好公司）
2. **不懂不买，而不是不买就亏了**
3. **市场短期是投票机，长期是称重机**（格雷厄姆名言）
4. **人生最好的投资是投资自己**，赚钱只是副产品
5. **不要为了投资失去睡眠**，仓位永远要在心理承受范围内
6. **保持谦逊**：市场经常让最聪明的人出丑
7. **坚持记录**：投资日记是最好的老师
8. **慢即是快**：复利需要时间，急于求成反而慢
